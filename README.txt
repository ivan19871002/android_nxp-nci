About
=====

This package contains the files allowing to patch AOSP source in order to 
add support for NXP-NCI NFC controllers to an Android KitKat based system.


Installation
============

Follow instructions from "AN11690 - NXP-NCI Android Porting Guidelines" document


History
=======
